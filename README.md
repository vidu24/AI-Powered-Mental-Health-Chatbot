
# 🧠 AI-Powered Mental Health Chatbot

An intelligent **AI-driven mental health chatbot** designed to analyze user emotions and provide **empathetic, context-aware responses** using Natural Language Processing (NLP).

---

## 📌 Project Overview

This project focuses on building a conversational AI system that can:

* Detect user emotions from text input
* Classify mental states such as **anxiety, depression, anger, fear, and joy**
* Provide supportive and meaningful responses
* Recommend helpful resources (e.g., videos, coping strategies)

The chatbot aims to act as a **first-level emotional support system**, improving accessibility to mental health assistance. ([GitHub][1])

---

## 🚀 Features

* 🧠 **Emotion Detection**

  * Classifies user input into emotional categories
* 💬 **Conversational AI**

  * Context-aware responses for better interaction
* 📊 **NLP-Based Processing**

  * Uses machine learning/NLP techniques for analysis
* 🎥 **Resource Recommendations**

  * Suggests relevant YouTube content for coping strategies
* 🌐 **User-Friendly Interface**

  * Simple interaction for ease of use

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/vidu24/AI-Powered-Mental-Health-Chatbot.git
cd AI-Powered-Mental-Health-Chatbot
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

Activate it:

* Windows:

```bash
venv\Scripts\activate
```

* Linux/Mac:

```bash
source venv/bin/activate
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```bash
python app.py
```
